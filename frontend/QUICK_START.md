# CodeJAM - Quick Start Guide

Get up and running with CodeJAM in just a few minutes!

## 🚀 3-Step Quick Start

### Step 1: Install Dependencies
```bash
cd codejam-project
npm install
```

### Step 2: Start Development Server
```bash
npm run dev
```

### Step 3: Open in Browser
Navigate to: **http://localhost:3000**

That's it! You're ready to code! 🎉

## 📝 First-Time Walkthrough

### 1. Register/Login
- Click "Register" on the landing page
- Enter any email and password (demo mode)
- You'll be redirected to the dashboard

### 2. Create Your First Project
- Click "+ New Project" button
- Enter a name: e.g., "My First Project"
- Select a language: e.g., "Python"
- Choose visibility: "Public" or "Private"
- Click "Create Project"

### 3. Write and Run Code
- Type your code in the editor
- Click "Save" to save your work
- Click "Run" to execute the code
- See output in the bottom panel

### 4. Manage Files
- Click the "+" icon in the file explorer to add files
- Click on any file to edit it
- Delete files by hovering and clicking the trash icon

### 5. Share Your Project
- Click the "Share" button
- Choose "Copy Link" to get a shareable URL
- Or choose "Email" to send an invitation
- Collaborators will request access when they open the link

## 💡 Tips & Tricks

1. **Auto-save**: Projects save automatically to browser storage
2. **Keyboard Shortcuts**: Use Ctrl+S (Cmd+S on Mac) to save
3. **Multiple Projects**: Create unlimited projects
4. **File Organization**: Use descriptive file names like `utils.py`, `helpers.js`
5. **Search**: Use the search box to filter projects by name or language

## 🔧 Common Tasks

### Create a New Project
Dashboard → "+ New Project" → Fill form → "Create Project"

### Add a File
Editor → File Explorer → "+" icon → Enter filename → "Create"

### Delete a File
Editor → File Explorer → Hover over file → Click trash icon

### Share a Project
Editor → "Share" button → Choose method → Copy/Send

### Switch Projects
Dashboard → Click on any project card
